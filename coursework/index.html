Systems Programming

Spidey and Thor

Contact me for the repository

The final project for my Systems Programming class was to create an HTTP web server in C called Spidey, accompanied by a client, written in Python, called Thor (our professor had an affinity for Marvel superheroes) which we used to test and benchmark our server. During this project we learned about TCP sockets, server-client architecture, CGI scripts, and parallelism: we implemented a forking mode so that the server could handle multiple connections simultaneously. Data Structures

ND Running Pathfinder

GitHub Repository

For our final project in Data Structures, our group created a pathfinding application designed to create a running route for the user. We used Notre Dame's own Map API to gather location data about campus buildings and landmarks, and used the Google Maps Distance API to build a graph (our own custom C++ class) of campus where edge weights represented the walking distance between nodes. We then implemented A* search as well as a Target-Value Search algorithm (a modified iterative-deepening breadth-first search, based on this paper) to generate a path from point A to point B of length X, where A, B, and X were parameters supplied by the user.

Fundamentals of Computing

Tetris

GitHub Repository

For my final project in Fundamentals of Computing, I created a simple Tetris clone. The game included a level system that gradually increased the difficulty as well as a scoring system. I used a custom library that was provided for the course that allowed for basic X11 graphics and keyboard input.
