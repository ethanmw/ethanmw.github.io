My Projects.

The Website

This website is my main side project. I've written and I host the site myself.

Code

The website's source code is stored in a GitHub repository here. I've written it using CSS from the Bootstrap 4 framework, along with my own custom CSS stylesheet. It was my first experience in web development; I taught myself HTML and CSS using a variety of online resources. I hand-wrote the site's code; in the beginning, I was learning HTML and wanted to jump in to creating the site without having to learn how to use a site generator like Hugo or Jekyll. I hope to expand the site in the future using one of these solutions to start a blog site.

Hosting

I host the website myself on a Digital Ocean VPS. I use Nginx to serve the site. I decided to go this route instead of using shared hosting because I wanted to learn something about how web servers work and how to manage a system.
